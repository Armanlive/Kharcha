# 💸 Kharcha - Premium Expense Tracker

![UI/UX](https://img.shields.io/badge/UI%2FUX-Premium_Dark_Mode-6366f1)
![Tech Stack](https://img.shields.io/badge/Tech-Vanilla_JS_|_HTML_|_CSS-f59e0b)

**Kharcha** is a lightweight, single-page progressive web application (PWA) designed to track daily expenses with a sleek, modern, and highly intuitive user interface. Built purely with Vanilla JavaScript, it requires no backend and stores all data securely in your browser's local storage.

🔗 **[Live Demo: Try Kharcha Here!](https://armanlive.github.io/Kharcha/)**

---

## ✨ Features

- **Premium Dark Mode UI**: A visually stunning, modern interface with soft gradients and clean borders.
- **Magic Entry (SMS Parser Simulation)**: Automatically fills in expense details by simulating the parsing of bank debit SMS messages—no manual typing required!
- **Interactive Analytics Dashboard**: Visualizes your spending habits instantly using **Chart.js** doughnut charts.
- **Month-over-Month Tracking**: Automatically calculates current vs. previous month expenses and displays percentage changes.
- **Professional PDF Reports**: Generates and downloads highly structured, invoice-style expense reports (complete with dot-dashed borders and footers) using **jsPDF** and **jspdf-autotable**.
- **Data Persistence**: Uses `localStorage` to ensure your financial data is safe even after you close the browser.

---

## 🛠️ Tech Stack

- **HTML5 & CSS3** (Custom variables, responsive grid, flexbox layout)
- **Vanilla JavaScript (ES6+)** (No bulky frameworks)
- **[Chart.js](https://www.chartjs.org/)** (For interactive data visualization)
- **[jsPDF](https://parall.ax/products/jspdf) & [jsPDF-AutoTable](https://github.com/simonbengtsson/jsPDF-AutoTable)** (For rendering custom PDF exports)
